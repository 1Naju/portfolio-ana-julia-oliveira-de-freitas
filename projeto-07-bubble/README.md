# 🔧 Gestão Low-Code com Bubble.io

## 📝 Descrição do Projeto
Este projeto consiste no desenvolvimento de uma aplicação web de gestão utilizando a **IA do Bubble.io** como acelerador, aplicando boas práticas de engenharia de software: modelagem prévia de dados, regras de privacidade (Privacy by Design), Option Sets, organização de workflows e documentação in-platform.

Desenvolvido como parte da disciplina de **Engenharia de Prompt e Aplicações em IA**, o projeto demonstra que a IA gera apenas um rascunho — a engenharia real começa onde a IA termina.

🔗 **Acesse o app:** [anajuliafreitas0407.bubbleapps.io/version-test](https://anajuliafreitas0407.bubbleapps.io/version-test/)

## 🚀 Tecnologias Utilizadas
* **Plataforma:** Bubble.io (Plano Spark)
* **IA utilizada:** Bubble AI
* **Banco de dados:** Bubble Data (Cloud)
* **Infraestrutura:** Bubble Hosting (HTTPS gratuito)

## 📊 Resultados e Aprendizados
* **Privacy by Design:** cada usuário enxerga apenas os próprios dados.
* **Option Sets:** todos os status definidos como conjuntos de opções, eliminando textos hardcoded.
* **Governança:** workflows organizados por cores e comentados com Notes.
* **Estratégia de saída:** dados exportáveis via Data API em JSON, permitindo migração para React/Node.js se necessário.

## 🔧 Como foi executado
1. Planejamento do banco de dados fora do Bubble.
2. Geração do blueprint com IA do Bubble.
3. Auditoria e correção da interface e workflows gerados.
4. Configuração das regras de privacidade.
5. Publicação e teste de isolamento de dados entre usuários.
