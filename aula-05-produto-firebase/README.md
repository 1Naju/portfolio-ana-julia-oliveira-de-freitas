# 🌑 ShadowLab — Produto Autoral com Firebase

## 📝 Descrição do Projeto
O **ShadowLab** é uma ferramenta de geração de design neumórfico desenvolvida como produto autoral a partir da engenharia reversa realizada na aula anterior. O objetivo foi transformar a referência original em um produto com identidade própria, nome comercial, novo visual e 4 diferenciais funcionais inéditos.

Desenvolvido como parte da disciplina de **Engenharia de Prompt e Aplicações em IA**, o projeto integra o Gemini como copiloto de IA e utiliza Firebase como infraestrutura.

🔗 **Repositório:** [github.com/1Naju/ShadowLab](https://github.com/1Naju/ShadowLab)
🔗 **AI Studio:** [Acessar projeto](https://ai.studio/apps/34baefa2-9529-433b-916b-8254692283bc)

## 🚀 Tecnologias Utilizadas
* **Frontend:** HTML5, CSS3 (Neumorfismo), JavaScript
* **IA integrada:** Google Gemini (via API)
* **Infraestrutura:** Firebase Hosting, Firebase Authentication
* **Plataforma de desenvolvimento:** Google AI Studio

## 📊 Resultados e Aprendizados

**4 Diferenciais Implementados:**
1. **Share via URL** — Gera um link com todas as configurações do design atual.
2. **IA sugere cor (Gemini)** — O usuário descreve um estilo e a IA sugere automaticamente a cor.
3. **Exportar CSS** — Baixa um arquivo `.css` pronto para uso no projeto.
4. **Favoritos** — Salva designs no navegador para reaplicar com um clique.

## 🔧 Como foi executado
1. Análise e mapeamento da ferramenta de referência.
2. Definição de nome, paleta neumórfica e diferenciais competitivos.
3. Desenvolvimento com Gemini como copiloto no AI Studio.
4. Deploy via Firebase Hosting com HTTPS gratuito.
